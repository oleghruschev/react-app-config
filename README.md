# react-app-config

#### react-app-config - это готовая конфигурация одностраничного приложения, которая содержит в себе:
- сборку js, jsx scss, img модулей с помощью webpack;
- конфигурацию babel для транспиляции кода;
- react-router;
- библиотека redux;
- redux-midleware.
- css-modules;
- конфигурацию eslint.
